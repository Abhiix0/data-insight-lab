# Data Insight Lab

A collection of data analysis projects built using Power BI, focused on exploring real-world datasets and transforming raw data into clear, actionable insights through visualization and analytical storytelling.

## Projects

### 1. Tech Layoffs Analysis

Explores global technology layoffs to uncover patterns in workforce reductions across companies, industries, and time periods.

### 2. Streaming Content Analysis

Analyzes streaming platform content to identify trends in genres, production countries, runtime patterns, and content growth over time.

### 3. Airline Flight Delay Analysis

Investigates airline operations data to understand delay patterns, airport congestion, flight performance, and operational efficiency.

## Tools & Skills

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation
* Data Visualization
* Insight Generation

## Goal

The goal of this repository is to demonstrate practical data analysis skills by working with real-world datasets and building dashboards that highlight meaningful trends and insights.
